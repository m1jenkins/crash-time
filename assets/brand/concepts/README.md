# Spur Auto ad-concept logo

This logo originated in an AI-generated OpenAI Ads concept. It is not the
current production website logo and should receive brand approval before it
replaces `assets/brand/spur-auto-logo.svg`.

## Files

- `spur-auto-concept-logo-on-navy.png` — exact 420×170 crop from the ad concept,
  with its original navy background.
- `spur-auto-concept-logo-reverse.png` — exact 420×170 concept logo with a
  transparent background; intended for navy or other dark backgrounds.
- `spur-auto-concept-logo-dark.png` — AI-adapted transparent version for white
  or light backgrounds. Its lettering and texture are close to, but not a
  pixel-exact reproduction of, the source concept.

At 2× display density, the exact 420-pixel-wide versions should be displayed no
wider than about 210 CSS pixels. A production vector redraw is recommended
before using the concept at larger sizes, in print, or as the permanent brand.

## Production record

The exact versions were extracted from the original ad-concept image without
redesigning the logo. The light-background adaptation was produced in built-in
image-edit mode with the source logo as its reference. The edit requested that
only the white lettering and rules change to deep navy (`#0d1b33`), while the
oxide-red and muted-sage mark remained unchanged. A flat magenta background was
then removed locally to produce transparency.
